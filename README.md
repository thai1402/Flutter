# Controle Financeiro Flutter

Aplicativo desenvolvido em Flutter para gerenciamento de receitas e despesas.

## Funcionalidades

- Cadastro de usuários
- Login
- Dashboard financeiro
- Adição de receitas
- Adição de despesas
- Cálculo automático do saldo
- Persistência local com SQLite

## Tecnologias

- Flutter
- Dart
- SQLite (sqflite)
